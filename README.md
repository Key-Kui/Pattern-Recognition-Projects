# Pattern Recognition Projects

Machine learning algorithms implementation for pattern recognition tasks.

## Projects

### SVM (Support Vector Machine)
- Classification using SVM with RBF kernel
- Bank marketing dataset classification
- ROC curve and confusion matrix visualization

### Bayesian
- Bayesian decision theory implementation
- Gaussian distribution-based classification

### Fuzzy
- Fuzzy pattern recognition using FCM clustering
- Membership function analysis

### ACO (Ant Colony Optimization)
- Path optimization algorithm
- Traveling salesman problem solution
- Convergence visualization

## Requirements

```bash
pip install numpy matplotlib scikit-learn
```

## Usage

```bash
# SVM
python SVM/SVM.py

# Bayesian
python Bayesian/Bayesian.py

# Fuzzy
python Fuzzy/Fuzzy.py

# ACO
python ACO/ACO.py
```
